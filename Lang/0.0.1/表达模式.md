#### 模板式
模板式在表达字符串方面具有非常好的效果

```
let Jop = "Jop"
let names = [$ Jackie Tom $Jop \$Mj ]

let str = "Jackie"

join($ hello $str)      // helloJackie
join($ hello   \ $str)  // hello   Jackie
join( "hello" str )     // helloJackie
```

#### 变量式

```
let Jop = "Jop"
let names = [ "Jackie" "Tom" Jop "$Mj" ]
```
