定义必须发生在使用之前
```
fn echo( str ) {
    println(str)
}

echo("Hello")
```
```
echo("Hello") // Error: "echo" 尚未定义

fn echo( str ) {
    println(str)
}
```

子域共享变量
```
{
    a := 74

    {
        print(a) // 74
    }
}
```
```
{
    {
        a := 74
    }
    print(a) // Error： "a" 尚未定义
}
```

动态作用域
```
fn[5] fns
for i:=0 i<5 i++ {
    fns[i] = () => {
        println(i)
    }
}

fn[3]() // 3
```
