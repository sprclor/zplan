经典多层函数调用方式

```
a(b(c(d(e(f())))))
```

使用同级方式

```
a | b | c | d | e | f
```



#### 多参数

```
fn oprA ( a b ) => a + b a * b
fn tripleo ( a b c ) => a * ( b - a / c ) * c
```

经典

```
tripleo( oprA( 3 5 ) 2 )
tripleo( 2 oprA( 3 5 ) )

let { a b } = oprA( 3 5 )
tripleo( a 2 b )
```

同级

```
tripleo( ; 2 ) | oprA( 3 5 )
tripleo( 2 ; ) | oprA( 3 5 )
tripleo( ; 2 ; ) | oprA( 3 5 )
```



#### 不定参数

```
fn triplilocial( ns ) {
	s := 0
	for n,i in ns {
        s += i % 2 == 0 ? n : ns[(i / 2) % 2] * n
	}
	return s
}
fn sum( ... ) {
	s1, s2 := 0
	s3, s4 := 1
	for n in @args[0] {
        s1 += n; s2 -= n
        s3 *= n; n4 /= n
	}
	return s1 s2 s3 s4
}
```

```
let { a b c d } = sum( 2 3 5 7 11 )
triplilocial( a c d b )
```

```
triplilocial | (a b c d) => ( a c d b ) | sum( 2 3 5 7 11 )
```
