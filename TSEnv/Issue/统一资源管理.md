## 引用

#### ID

资源在其作用域中具有唯一的ID，使用`R`资源对象进行资源的引用

```
type R ( path ) {
    ID ResourceID
    type public/private
    path Path
}

struct {
    R( shop/name.png )
}
```

```
type Icon like R {
    @ {
        state {
			normal
			danger
			light
			flat
			stroke
			fill
        }
    }
}
```
