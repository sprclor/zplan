状态在变更时会执行经历一个过程，这个过程叫变更中。变更中的状态有两种，`添加时`与`移除时`。

#### 选择器

```
box {
    [hover] { // 普通状态
        center sin(time) * 12px, cos(time) * 20px
    }
    [+hover] { // 添加hover状态时
    	scaling 2, .5s
    }
    [-hover] { // 移除hover状态时
        
    }
    [*hover] { // 选择+和-，相当于 [ +hover -hover ]
        
    }
}
```
